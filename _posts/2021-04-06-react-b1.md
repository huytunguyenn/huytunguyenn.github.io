### **Stateful Component**

- nor: mỗi lần state thay đổi thì phải redraw lại giao diện.

⇒ React ko cần redraw ⇒ nhờ ma thuật `setState`

- khi React thấy `setState` thay đổi ⇒ tự động gọi `render()`

```jsx
class Timer extends React.Component {
  constructor(props) {       
    super(props);    // goi ham dung cua lop cha (tuc la truyen props vo constructor cua React.Component
    this.state = { seconds: 0 }; // trang thai dau tien gan = 0
  }

  tick() {  // moi giay + len 1
    this.setState(state => ({
      seconds: state.seconds + 1 
    }));
		// ko can redraw :3
  }

  componentDidMount() { // moi giay goi ham tick 1 lan
    this.interval = setInterval(() => this.tick(), 1000); 
  }

  componentWillUnmount() {  // ngan chan memory leak
    clearInterval(this.interval); 
  }

  render() {
    return (
      <div>
        Seconds: {this.state.seconds}
      </div>
    );
  }
}

ReactDOM.render(
  <Timer />,
  document.getElementById('timer-example')
);
```

**Thay đổi trạng thái**: 

 Lớp cha thay đổi đc props lớp con.

- `this.props` ⇒ read-only *(thật ra sửa đc nhưng k có ý nghĩa j hết)*
- `this.state` ⇒ mutable
